# Cricsheet-Match-Analysis_Guvi_project
Mini project about Cricket Match Analysis based on below sources
selenium,
pandas,
requests,
matplotlib,
seaborn
by using the above code languages we are going to analyse the cricket match data,By using EDA how many bruns,best batsman,bowler,winning matches
Matplotlib and seaborn are used to generate a plots and histogram of the match data

selenium is used to handle the web browser to download the Json files.
Pandas are used for data frame to convert the Json data into df to csv to generate the plot
